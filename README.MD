## 📱 简介

**Martis-eBillboard** 是一款专为信息发布场景设计的 Android 应用，利用系统原生 WebView 框架实现网页内容的全屏展示。软件精简高效，兼容广泛，可作为系统 Launcher 使用，实现无人值守的信息展示。

---

## ✨ 核心特性

### 1. 无缝对接 B/S 架构
- 直接加载网页内容，支持网页信息展示和交互
- 兼容各类 Web 技术栈（HTML5、CSS3、JavaScript）
- 支持设置自定义 User-Agent

### 2. 精简高效
- 基于系统原生 WebView 框架，软件体积小巧
- 资源占用低，运行流畅
- 无冗余功能，专注信息展示

### 3. 广泛兼容性
- **最低支持 Android 4.4（API 19）**
- 支持新旧软硬件设备
- 适配低配置硬件平台

### 4. Launcher 霸屏模式
- 可作为系统默认 Launcher 使用
- 开机自动启动，全屏显示
- 防止用户退出，适合公共展示场景

### 5. 密码保护
- 设置页面支持密码保护（默认密码：123）
- 防止设置被随意篡改
- 支持密码修改功能

### 6. 运维功能
- 系统设置访问
- 已安装应用列表
- 浏览器 UA 设置
- 自动加载网址配置
