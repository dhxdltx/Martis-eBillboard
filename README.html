🔨 如何构建 APK

安装 Android Studio

下载 Android Studio（免费）并安装 SDK，选择 API 14~33 的组件。

导入项目

打开 Android Studio → File → Open → 选择 WebViewLauncher 文件夹

等待 Gradle 同步

首次同步会自动下载依赖，需要网络连接，约 1~5 分钟。

构建 Debug APK

菜单 Build → Build Bundle(s) / APK(s) → Build APK(s)，完成后点击 "locate" 定位 APK 文件。

# 或者命令行构建（需要配置好 JAVA_HOME 和 Android SDK）
./gradlew assembleDebug
# APK 位置：app/build/outputs/apk/debug/app-debug.apk

安装到设备

启用开发者选项 → USB 调试，连接手机后执行：

adb install app/build/outputs/apk/debug/app-debug.apk

设置为默认启动器（可选）

在系统提示"选择默认桌面应用"时，选择 WebView Launcher 并点击"始终"。

💡 使用提示

返回键操作：在主界面按返回键会弹出快捷菜单（刷新 / 主页 / 设置 / 退出），不会直接退出应用。网页有历史记录时，返回键会先导航到上一页。
进入设置：长按 WebView 空白区域 → 点击菜单按钮（右上角三点）→ 设置；或在返回键弹窗中选择"设置"。
注意：如果目标网站使用 HTTP（非 HTTPS），已在 AndroidManifest.xml 中开启 android:usesCleartextTraffic="true"，以确保 Android 9+ 也能正常加载 HTTP 页面。如果安全性要求高，可将此项改为 false。
自定义图标：将 PNG 格式的图标文件分别放入 res/mipmap-mdpi（48×48）、hdpi（72×72）、xhdpi（96×96）、xxhdpi（144×144）目录，命名为 ic_launcher.png。
